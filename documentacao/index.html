Corretor Ortográfico - Informações Gerais

  • Linguagem utilizada: C
  • Utilidade: Avaliação de textos e correção de palavras possivelmente erradas.

main.c

Arquivo responsável pela interação com o usuário.

Visão geral

O main.c é responsável por:

  • Importação do dicionário de palavras
  • Interação com o usuário por meio de argumentos ou interface em linha de comando
  • Retorno e/ou reescrita do texto corrigido

O funcionamento da aplicação pode ser visto de forma detalhada no README.md do repositório do projeto no GitHub.

funcoes.h

Arquivo de cabeçalho com funções e implementações de algoritmos úteis para a aplicação.

int split(char str[], char **array)

Separa palavras de uma string e as coloca em um array de destino, retornando um inteiro que condiz com o número de palavras da string.

Parâmetros

  • char str[] - String a ter as palavras separadas
  • char **array - Array de destino para as palavras separadas

void ordena(char *array[], int inicio, int fim)

Ordena array de palavras com a implementação do Merge Sort.

Parâmetros

  • char *array[] - Array a ser ordenado
  • int inicio - Posição inicial do array
  • int fim - Posição final do array

void merge(char *array[], int inicio, int meio, int fim)

Função com implemetação do algoritmo merge, utilizado para ordenação.

Parâmetros

  • char *array[] - Array a ser ordenado
  • int inicio - Posição inicial do array
  • int meio - Posição mediana do array
  • int fim - Posição final do array

void slice(char *dest[], char *array[], int inicio, int fim)

Função que copia array em uma posição delimitada para um outro array de destino.

Parâmetros

  • char *dest[] - Array de destino
  • char *array[] - Array de origem
  • int inicio - Posição inicial de cópia para array de destino
  • int fim - Posição final de cópia para array de destino

int busca(char *array[], char item[], int inicio, int fim)

Função com implementação do algoritmo de busca binária.

Parâmetros

  • char *array[] - Array de strings no qual acontecerá a busca
  • char item[] - String a ser buscada no array
  • int inicio - Posição inicial do array de busca
  • int fim - Posição final do array de busca

char *correcao(char **dicionario, int nDicionario, char str[])

Função que avalia palavra passada, procurando-a no dicionário e efetuando a busca por palavras parecidas caso essa não seja encontrada. Retorna uma string correspondente a avaliação da palavra.

Parâmetros

  • char **dicionario - Array com o dicionário de palavras importado
  • int nDicionario - Inteiro com o tamanho do dicionário
  • char str[] - String a ser avaliada

void getPalavra(char palavra[])

Função que limpa caracteres não alfabéticos de uma palavra passada.

Parâmetros

  • char palavra[] - String a ter caracteres limpos

char *corrigePalavra(char str[], char correta[], char errada[])

Função que corrige a palavra a ser inserida novamente no texto de acordo com a formatação inicial da entrada. Retorna a string reformatada.

Parâmetros

  • char str[] - String dada na entrada
  • char correta[] - String correspondente a palavra corrigida
  • char errada[] - String correspondente a palavra errada

int buscaParecidas(char **array, int n, char palavra[], int enc[])

Algoritmo para encontrar palavras parecidas a palavra passada como parâmetro no array de palavras com base no percentual de caracteres iguais. Retorna o índice da palavra mais próxima encontrada.

Parâmetros

  • char **array - Array de strings contendo um conjunto de palavras
  • char palavra[] - String correspondente a palavra a ter uma próxima buscada
  • int enc[] - Array com inteiros correspondentes aos índices de palavras parecidas anteriormente encontradas

int existeNoArray(int array[], int index)

Verifica a existência de um elemento em um array, retornando 0 caso não encontre e 1 caso seja encontrado.

Parâmetros

  • int array[] - Array de inteiros
  • int index - Inteiro a ser buscado no array

void invertePalavra(char palavra[], char ret[])

Função que inverte a ordem dos caracteres de uma string as colocando no array de destino.

Parâmetros

  • char palavra[] - String a ser invertida
  • char ret[] - Array de destino

char **lerLinhas (char *fileName, int *n)

Parâmetros

  • char str[] - String referente ao elemento do nó a ser adicionado

void imprimeLista(struct node *l)

Parâmetros
